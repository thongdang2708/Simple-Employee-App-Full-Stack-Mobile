Please download a React Native Debugger through a link:

```bash
https://github.com/jhen0409/react-native-debugger
```

Run backend side:

```bash
cd backend
```

```bash
docker network create employeeapp
```

```bash
docker-compose up
```

When stopping the app, close with the below command in backend directory:

```bash
docker-compose down
```

Checking the running containers, use the command in backend directory:

```bash
docker-compose ps
```

Run front-end side:

```bash
cd frontend
```

```bash
npm install
```

```bash
expo start
```
